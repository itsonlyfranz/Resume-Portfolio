A modern NFT marketplace that lets artists upload, mint, and sell digital assets on the Polygon blockchain — with an admin-gated approval pipeline, automated OpenSea listings, and live cryptocurrency pricing.

## Architecture

- **Next.js App Router** with server-side rendering for optimized performance
- **Supabase** for authentication, PostgreSQL database, and file storage
- **ethers.js v6** for ERC-721 minting and blockchain interactions on Polygon
- **opensea-js** for automated NFT listings with WETH pricing
- **OpenSea Stream API** for real-time sale event tracking
- **CoinGecko API** for live PHP to WETH exchange rates (60-second cache)
- **MetaMask SDK** for wallet connection and Polygon network switching

## Key Flows

**Mint flow:** Seller uploads artwork with PHP price → mint request created → admin approves in dashboard → NFT minted on Polygon → token ID stored in Supabase.

**Sell flow:** Seller creates sell request → admin approves → PHP price converted to WETH at live rate → NFT automatically listed on OpenSea → Supabase pg_cron polls OpenSea every 5 minutes → sold event triggers status update and email notification via Resend.
