An AI-powered insurance chatbot that answers product questions from a markdown knowledge base and guides users through structured insurance quotation flows for auto, home, and life insurance.

## Architecture

The system uses a **LangGraph StateGraph** to orchestrate multiple specialized nodes:

- **detectIntent** — classifies each message as quotation or conversational, with mid-quote side-question escape routing
- **rewriteQuery** — rewrites user queries for better retrieval
- **retrieveChunks** — vector similarity search over the knowledge base
- **gradeChunks** — LLM-based relevance grading with retry loop
- **askClarifyingQuestion** — branded clarification when retrieval fails
- **finalizeResponse** — the sole node that emits assistant messages, supporting RAG answers, citations, and quote CTAs

## Key Features

- Stateful quotation flow with field-by-field collection and validation
- Mid-quote informational side questions route to RAG, preserving quote progress
- Paused quote context restored when user continues
- Greeting and low-context detection with branded ShieldBase onboarding response
- Groq `llama-3.3-70b-versatile` for chat and structured classification; OpenRouter for embeddings
